# OpenQuiz AI 🧠🎙️

OpenQuiz AI is an open-source, modern vocabulary learning platform built with **Next.js 16**, **Supabase**, and **Groq Llama 3.3 AI**. It combines the efficiency of Spaced Repetition (SRS SM-2) with an immersive 3D Flashcard experience and real-time AI pronunciation feedback.

## 🚀 Key Features

*   **🃏 3D Flashcards**: A beautifully crafted, glassmorphic 3D flip card UI for engaging learning sessions.
*   **🧠 Spaced Repetition (SRS SM-2)**: Smart algorithm that determines the optimal time for you to review a word so it moves into your long-term memory.
*   **🎙️ AI Pronunciation Trainer**: Practice speaking with real-time feedback. It uses Groq API (Llama 3.3) and the browser's native Speech Recognition to grade your accuracy, prosody, and give tailored advice.
*   **✨ AI Auto-Fill Generator**: Just type a word, and the AI will automatically fetch its IPA, definition, Vietnamese translation, context example, and synonyms.
*   **🔐 Authentication & Database**: Built securely with Supabase Auth, Row Level Security (RLS) policies, and a PostgreSQL database.
*   **🔥 Streak System**: Keep up the momentum with an automated daily learning streak tracker.
*   **🎓 Dynamic Ecosystem**: Complete learning tools covering Roleplay conversations, Grammar practice, Reading comprehension, Dictation, and Quizzes with AI integrations tailored to your current Band level.

## 🛠️ Tech Stack

*   **Framework**: Next.js (App Router, Server & Client Components)
*   **Database & Auth**: Supabase (PostgreSQL)
*   **AI Integration**: Groq API (Llama-3.3-70b-versatile)
*   **Styling**: Tailwind CSS + Custom CSS (Glassmorphism, 3D Transforms, Glows)
*   **Icons**: Lucide React
*   **Animations**: Framer Motion
